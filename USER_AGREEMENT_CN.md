#### 2. 身份验证与隐私安全
为了识别您的上传内容并防止恶意刷票，本服务需要进行基础的身份验证：
*   **加密凭证：** 系统将基于您的 **Steam ID** 通过加盐哈希 (Salted Hash) 算法生成一个唯一的**玩家密钥 (User Key)**。此加密不可逆，无法被逆向推导出原始 Steam ID。
*   **知情同意：** 虽然我们对凭证进行了加密处理，但请知悉 Steam ID 本质上属于公开信息。
*   **用途限制：** 该密钥仅用于验证上传者身份、记录点赞/点踩数据以及管理您上传的蓝图。
*   **隐私承诺：** 我们不会将您的 IP 地址与您的游戏身份公开关联，且仅在维护服务器安全（如防御攻击、封禁违规用户）时记录和使用 IP 地址。

为了维护社区的良好环境，若系统检测到或经举报核实存在以下违规行为，服务器将**拒绝您的访问请求 (Access Denied)**：

*   **依赖提醒：** 上传蓝图时，请确保文件是有效的游戏数据。如果蓝图包含其他模组的物品，请尽量在描述中注明。
